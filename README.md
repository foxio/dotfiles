## Installation
```
cd; curl -#L http://dotfiles.fox.io | tar -xzv --strip-components 1 --exclude={README.md,LICENSE.md,.gitignore}; sh $HOME/setup.sh
```

## Clean Up

Once the above command is run, you can safely delete `setup.sh`.

```
rm $HOME/setup.sh

```
